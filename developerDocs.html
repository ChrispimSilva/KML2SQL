KML2SQL Nuget Documentation.

Getting Started

First, we need to install the NuGet package:

Install-Package Kml2Sql

The bare minium code to insert a KML file into SQL Server would then look like this:

var fileStream = File.Open("myFile.kml", FileMode.Open);
var mapper = new Kml2SqlMapper(fileStream);
using (var connection = new SqlConnection("myConnectionString"))
{
    connection.Open();
    var createTableCommand = mapper.GetCreateTableCommand(connection);
    createTableCommand.ExecuteNonQuery();
    foreach (var mapFeature in mapper.GetMapFeatures())
    {
        var command = mapFeature.GetInsertCommand();
        command.Connection = connection;
        command.ExecuteNonQuery();
    }
}

Now let's check SQL

Select * 
From UsZips 
Where ZCTA5CE10 > '27006' --This is the zip code
AND ZCTA5CE10 < '28909'   --range for North Carolina

It's as easy as that. More documentation Coming SoonTM, but in the mean time, all of the customization/configuration can be done via the Configuration object like so: 

var mapper = new Kml2SqlMapper(fileStream);
mapper.Configuration.IdColumnName = "Id";
mapper.Configuration.PlacemarkColumnName = "Shapes";
mapper.Configuration.TableName = "UsZips";
mapper.Configuration.MapColumnName("XmlNodeName", "ColumnName");
